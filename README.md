# LCare IQ — النشر على Netlify

منصة متابعة مرضى السكري مع ذكاء اصطناعي محمي بخادم آمن.

## 📁 محتويات المجلد

```
lcare-netlify/
├── index.html                    ← الموقع (الواجهة)
├── netlify.toml                  ← إعدادات Netlify
├── netlify/
│   └── functions/
│       └── claude.js             ← الخادم الآمن (يحمي مفتاح API)
└── README.md                     ← هذا الملف
```

---

## 🚀 خطوات النشر (طريقتان)

### الطريقة الأولى: السحب والإفلات (الأسهل)

> ⚠️ هذه الطريقة لا تفعّل الدالة (Functions). للذكاء الاصطناعي استخدم الطريقة الثانية.

1. ادخل إلى [app.netlify.com/drop](https://app.netlify.com/drop)
2. اسحب **محتويات** هذا المجلد (مو المجلد نفسه)
3. الموقع ينشر فوراً

### الطريقة الثانية: عبر GitHub (موصى بها — تفعّل الذكاء الاصطناعي) ✅

1. أنشئ مستودع (repository) جديد على GitHub
2. ارفع كل محتويات هذا المجلد إليه
3. ادخل إلى [app.netlify.com](https://app.netlify.com) → **Add new site** → **Import an existing project**
4. اختر **GitHub** ثم المستودع
5. اترك الإعدادات كما هي (netlify.toml يضبط كل شي) → **Deploy**

---

## 🔑 إضافة مفتاح API (خطوة إلزامية للذكاء الاصطناعي)

بعد النشر:

1. في لوحة Netlify، افتح موقعك
2. **Site configuration** → **Environment variables**
3. اضغط **Add a variable** → **Add a single variable**
4. الاسم (Key): `ANTHROPIC_API_KEY`
5. القيمة (Value): الصق مفتاحك الجديد من [console.anthropic.com](https://console.anthropic.com/settings/keys)
6. احفظ
7. **مهم:** أعد النشر — **Deploys** → **Trigger deploy** → **Deploy site**

> المفتاح يبقى على خادم Netlify فقط. لا يظهر في المتصفح ولا في كود الموقع إطلاقاً.

## ⚠️ ملاحظات مهمة

1. **البيانات محلية:** بيانات المرضى تُحفظ في متصفح كل جهاز (localStorage). لا توجد مزامنة بين الأجهزة بعد. صدّر JSON كنسخة احتياطية دورياً من داخل التطبيق.

2. **للمزامنة بين الفريق:** ستحتاج قاعدة بيانات سحابية (مثل Supabase). هذه خطوة لاحقة.

3. **التكلفة:** Netlify مجاني للاستخدام الخفيف. تدفع فقط مقابل استهلاك Claude API حسب الاستخدام.

4. **اختبار الذكاء الاصطناعي:** بعد ضبط المفتاح وإعادة النشر، جرّب ميزة "فحص أكلة" أو "تحليل" داخل ملف مريض. إذا ظهر خطأ، تأكد أن اسم المتغير بالضبط `ANTHROPIC_API_KEY` وأنك أعدت النشر.
